# Netflix-Themed Analytics Dashboard

A responsive, Netflix-themed analytics dashboard built with plain HTML/CSS/JS and Chart.js. Uses the provided `netflix_show_reviews.json` as the data source.

## Features
- Bar, Pie, and Line charts (Chart.js)
- Search and Rating filter controls
- KPI cards: Total Shows, Average Rating, Highest Rated
- Responsive layout with Bootstrap and custom Netflix-themed styles
 - Clickable red decade badges to filter by release decade
 - Top 5 shows panel showing highest-rated titles
 - Enhanced Netflix color accents and eye-catching KPI cards
